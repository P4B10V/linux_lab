My first day I decided to install 2 virtual machines. I went for Debian 13, one in cli mode and the other one to use it as admin. 

For that, the first thing i wanted to do was activate in each machine an adapter to use it as host-only and from that, install in the server `openssh-server`

In the admin mv i generate a pair of keys with `ssh-keygen` and copy it the .pub in the server with `ssh-copy-id` 

To do a bit of hardening on server side, i changed:
  - Port of ssh
  - Disable root login (i think is disable by default)
  - disable password login
  - enable only for publickey

i had to be very cautios because (it wont matter because i could just go to the mv server) i didnt want to get kicked and not being able to do ssh to the server. I wanted to do this way because in the future maybe i'll use a VPS and in those enviroments i would have to know what i have to do to prevent that

once i configured and verified everyhing from ssh i wanted to check the logs but i could find any in /var/log so i had to do a little research to learn that in debian 13 all the logs are in journalctl, good to know
