Today is the starting point of my project. First of all, I have to create the enviroment to do so. 

For that I'm going to install two virtual machines, one Debian CLI as server and another Debian as admin to manage the server.

The first service that I'm going to install is `openssh-server` to generate an SSH key par on my admin vm with `ssh-keygen` and then transfer it to the server using `ssh-copy-id` 

I also applied some basic SSH hardening on the server side, such as:
- Changed default SSH port
- Verify that root login is disable
- Disable password authentication
- Allowed only public-key authentication

Worth mentioning that I had to be very careful doing this configuration because one mistep and I would get locked out from my server (In this case is not a big deal because I can just log in directly to the server, but I wanted to do imitating a VPS scenario)

After finishing the SSH configuration, I wanted to check logs about the connections that I did but I couldn't find anything at all. So I had to do some research and discover that supposedly Debian 13 uses journalctl to check logs instead of the tradicional log files in /var/log
