I'm going to prepare now a new server starting with Samba. Also, I'm going to add two VHD to create in the future shared folders.

First of all, I want to do the right thing and configure LVM in the server 

With a 40 GB VDH my configuritation was:
 - Primary partition 1GB for /boot
With the remaining space I configured a LVM, making first a group volume (vg_dc) and then lv_root for /, lv_home, lv_swap and lv_var


My objective was RAID 1 + LVM, so I had to download `mdadm` to create first a raid:

First step: Create primary partition with 40G with `fdisk`
Second step: Configure raid: `mdadm --create /dev/md0 --level=1 --raid-devices=2 /dev/sdb1 /dev/sdc1`
Third step: Configure LVM creating physical volume -> volume group -> logical volume 
```
pvcreate /dev/md0 
vgcreate vg_data /dev/md0
lvcreate -l 100%FREE -n lv_data vg_data
```

Necessary to update system and detect raid on boot:
```
mdadm --detail --scan >> /etc/mdadm/mdadm.conf
update-initramfs -u
```

Create and mount share folder:
```
mkdir /srv/data
mount /dev/vg_data/lv_data /srv/data
```
And then configure auto-mount, `blkid /dev/vg_data/lv_data` to take id and put it in `/etc/fstab`, to verify `mount -a`
